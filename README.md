# the-snake

## Описание

the-snake — это игра «Змейка», разработанная на языке Python с использованием библиотеки pygame. Цель игры — управлять змейкой и стараться съесть как можно больше яблок. После каждого съеденного яблока змейка увеличивается в длине. Игра заканчивается при столкновении с собственным телом.

## Функциональность

- Управление змейкой с помощью клавиш со стрелками.
- Случайное появление еды на игровом поле.
- Увеличение длины змейки после поедания еды.
- Завершение игры при столкновении с самим собой.

## Установка

Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

Запустите игру командой:
```bash
python the_snake.py
```
